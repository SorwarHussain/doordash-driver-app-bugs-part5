# doordash-driver-app-bugs-part5
When attempting to start a dash in the **DoorDash Driver (Dasher) app**, the app fails to begin the session and displays an error message: **“Couldn't create dash, something went wrong.”** As a result, the driver is unable to start dashing.

# Bug Report
**Title:** Unable to start dash — “Couldn't create dash, something went wrong” error displayed


## Steps to Reproduce
1. Open the **DoorDash Driver (Dasher)** app.
2. Log in with a valid driver account.
3. Attempt to start dashing by selecting a zone and tapping **“Dash Now”**.
4. Observe the error message.

---

## Expected Behavior
- The dash session should start successfully.
- The app should transition to the active dashing screen.

---

## Actual Behavior
- An error message is displayed: **“Couldn't create dash, something went wrong.”**
- The dash does not start.
- The driver remains unable to dash.

---

## Frequency
- Occurs once.

---

## Environment
- **Application:** DoorDash Driver (Dasher) app  
- **Platform:** Android 
- **App Version:** 8.61.6
- **Date Observed:** Jan 6, 2026 
- **Network:** Mobile data

---

## Impact
- Completely blocks drivers from starting a dash.
- Prevents access to deliveries and earning opportunities.
- Forces repeated attempts or app restarts.
